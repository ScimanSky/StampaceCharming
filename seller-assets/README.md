# Seller Assets

This folder contains seller-side marketing outputs, not buyer-facing template files.

## Contents

- `listing-slides/`: PNG slides ready for Etsy listing images
- `social-crops/`: square crops for reuse in social previews or alternate marketplaces

## How to rebuild

```bash
npm run build:etsy
```

That command refreshes:

- `dist-etsy/guest-guide-atelier-template/`
- `dist-etsy/guest-guide-atelier-template.zip`
- `seller-assets/listing-slides/`
- `seller-assets/social-crops/`
