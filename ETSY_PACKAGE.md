## What stays out of the buyer ZIP

- seller checklists
- listing copy drafts
- listing asset source files
- build scripts
- repository tooling
- `node_modules`

## Listing clarity requirements

State these points in the Etsy listing:

- digital download only
- no physical item shipped
- hosting not included
- basic edits can be done in the browser customizer
- this is a website template, not a PDF
- ideal for Airbnb hosts, vacation rentals, villas and boutique apartments
