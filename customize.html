Publish

Generate the exact link you will send to guests

Add the public guide URL once the page is live. The template generates a guest-safe link automatically, and the QR printables use that guest version by default unless you enter a custom QR destination.

`Copy guest link` is the safest option to send in Airbnb messages, WhatsApp, email or SMS. Preview uses browser storage. Download `config.json` when the guide is ready, then replace the root file before publishing or deploying.
